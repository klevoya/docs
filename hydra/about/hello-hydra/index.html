Hello Hydra

Hello Hydra is an example playground to get familiar with Hydra. It consists of a standard eosio.token contract and a simple test case for it.

git clone git@github.com:klevoya/hello-hydra.git

After installing the dependencies (npm install) and logging into Hydra, the test in tests/token.test.js can be executed:

npm test

The test creates a token, issues some tokens to a user, who in turn transfers it to a second user. It should result in all tests passing:
